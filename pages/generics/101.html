Go Generics 101

This book talks about all aspects of Go custom generics. The contents mainly include

  • concepts and syntax of Go custom generics.
  • how to use Go custom generics.
  • the restrictions in the current implementation of Go custom generics.

The book is Go 1.20 ready now (update history).

(Please follow Go 101's official Twitter account @go100and1 to get latest Go 101 news and learn all kinds of Go knowledge.)
